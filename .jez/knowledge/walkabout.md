# Walkabout — the app demos itself

[Walkabout](https://github.com/jezweb/walkabout) gives a web app three connected capabilities: a guided voice tour that walks the real pages with a narration-synced moving spotlight, an ask-the-app AI guide whose question log doubles as the product roadmap, and one-command narrated demo videos (promo, training, socials) re-recordable whenever the app changes. The goal: the owner never gives a demo again.

Reach for it when an app needs onboarding tours, in-app help, or demo and training videos. It ships as a Claude Code plugin with working templates (real files from a shipped app, not scaffolds): point your agent at the repo, or install it so the skill loads itself:

```
/plugin marketplace add jezweb/walkabout
/plugin install walkabout
```
