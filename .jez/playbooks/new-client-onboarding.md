# New client onboarding

> Example showing the shape (no frontmatter needed, playbooks are usually just steps). Copy it for a real one; delete the sample once you've got the idea.

When to use: a new client has signed and you're setting them up.

1. Create their page in `clients/`. *Check:* it has their contacts and what you're doing for them.
2. Send the welcome email and intake form. *Check:* they've replied with the details you need.
3. Set up their tools and access (shared drive, billing in Xero, whatever applies). *Check:* you can see them in each system.
4. Book the kickoff call. *Check:* it's in the calendar and they've accepted.
5. Note anything unusual on their client page. *Check:* the next person could run this account cold.

Client-specific quirks go on the client's own page, e.g. [[acme-digital]].
